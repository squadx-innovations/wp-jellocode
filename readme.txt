== Description ==

[Demo](https://ide-dashboard.squadx.io) - [GitHub](https://github.com/squadx-innovations/wp-jellocode)

Jellocode is platform that allow authors to embed coding playground on web pages for an engaging experience.

== Features ==

JelloCode comes with the following features:

* Embed coding playground for engaging experience
* Visualize live compiled output
* Get user analytics on your playground
* Users collaborate on code through Github


== Usage ==

Iframes can be generated by creating a project and adding code files. This embedded code (iframes) can be inserted in the editor and the coding playground would be visible on the webpage.

== Installation ==

Installation from the WordPress admin.

1. Log in to the WordPress admin and navigate to *Plugins > Add New*.
2. Type *Jellocode* in the Search field.
3. In the results list *Jellocode* plugin should appear, click **Install Now** button.
4. Once it finished installing, click the *Activate* button.
5. That's it, now you can create your first project and start embedding the coding playground at your post pages.
